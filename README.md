
# 🎬 Proyecto Integrador: Animación Frame por Frame

## 🎥 Animación en Blender 

Este repositorio contiene el resultado de mi proyecto integrador para la materia de **Graficación**. El trabajo consistió en el desarrollo de una animación técnica realizada cuadro por cuadro, aplicando principios de movimiento y renderizado digital.

---

## 📖 Descripción del Proyecto

El objetivo fue crear una secuencia animada controlando manualmente la línea de tiempo. Se trabajó en la disposición de elementos en el espacio tridimensional y en la gestión de fotogramas clave para lograr una narrativa visual fluida.

### Características principales:

* **Keyframing:** Gestión manual de fotogramas para posición, rotación y escala.
* **Timing y Spacing:** Control del ritmo de la animación para lograr naturalidad en el movimiento.
* **Renderizado:** Configuración de la salida final para optimizar la calidad visual del video.
* **Composición:** Manejo de cámaras y encuadres para resaltar la acción de la animación.

## ⚙️ Uso y Requisitos

Para abrir y explorar el proyecto original, sigue estos pasos:

1.  **Software:** Tener instalado **Blender** (Versión 3.0 o superior).
2.  **Archivo:** Descargar el archivo `.blend` incluido en este repositorio.
3.  **Visualización:** Una vez abierto, presionar `Space` para reproducir la animación en el visor o navegar por el *Timeline* para ver los frames.
